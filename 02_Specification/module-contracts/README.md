# Module Contracts

Эта папка содержит **публичные API всех модулей** проекта. Каждый модуль имеет свой контракт-документ, описывающий:

- За что модуль отвечает.
- Что экспортируется наружу через barrel-файл `index.ts`.
- От чего модуль зависит (другие модули, библиотеки).
- Какие side-effects возможны.
- Как тестировать.

## Правило обновления

**Любое изменение публичного API модуля требует обновления его контракт-документа в том же PR.** Это касается:
- Добавления / удаления / переименования экспортов.
- Изменения сигнатур функций или классов.
- Изменения структуры экспортируемых типов.
- Изменения side-effects (например, появление нового storage-key).

Если PR меняет публичный API без обновления контракта — он автоматически incomplete и не должен быть merge-нут.
