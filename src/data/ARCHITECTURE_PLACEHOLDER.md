# src/data — Bundled reference data

Bundled JSON-ресурсы с опорными значениями (например, `b787-8-dry.json` с таблицей crosswind-лимитов).

Каждый JSON версионируется полем `dataVersion`. Изменение значений → инкремент версии → новый PR.

В MVP: `crosswind/b787-8-dry.json` (см. `04-domain-model.md` и `05-crosswind-algorithm.md`).
